## Designing
Drag and drop numbered blocks from the supply onto the grid.

Drag a block out of the grid to return it to the supply.

There is only one of each numbered block.

The blank block can be used as many times as you like.

## Colors
Double-click a block to bring up a list of available colors, select one to change the block's color.

## Capturing
When you have a design you like, click the "capture" button.

This will put the code of your design onto your clipboard.

## Loading
To load someone else's design, paste their code into the box and click "load".

## Clearing
The reset button will clear the design after your confirmation.
